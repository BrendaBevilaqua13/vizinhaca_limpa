Relatório de Validação – Projeto “Vizinhança Limpa”
1. Introdução

Este relatório apresenta o processo de validação do sistema de denúncias urbanas “Vizinhança Limpa”, desenvolvido como solução tecnológica para facilitar o registro e acompanhamento de denúncias relacionadas a lixo irregular e outros problemas ambientais no bairro.

A validação teve como objetivo comprovar a aplicabilidade, usabilidade e importância do sistema junto a um grupo real de moradores que vivenciam diariamente problemas ambientais na comunidade.

2. Objetivo da Validação

A validação buscou:

Verificar se o sistema atende às necessidades reais dos moradores.

Avaliar a usabilidade da interface por usuários leigos em tecnologia.

Testar o fluxo completo: cadastro → login → envio de denúncia → visualização.

Identificar melhorias necessárias antes da entrega final do produto.

3. Público-Alvo da Validação

A validação foi realizada com membros da Comunidade Unida, grupo pertencente à Igreja Nossa Senhora do Carmo, localizada no bairro Alto da Mangueira – Maracanaú/CE.

Esses moradores realizam ações ambientais e comunitárias no bairro, como limpeza de áreas comuns, monitoramento de pontos de descarte irregular e mobilização de vizinhos.

4. Metodologia de Validação

A validação foi realizada por video chamada, seguindo as etapas abaixo:

4.1. Apresentação do Sistema

Foi apresentada uma explicação sobre o propósito e funcionamento do sistema “Vizinhança Limpa”.

4.2. Demonstração Prática

Os participantes visualizaram as funcionalidades principais:

Login e criação de conta

Tela de criação de denúncia

Upload de informações

Acompanhamento das denúncias enviadas

4.3. Teste Prático pelos Participantes

Cada participante utilizou o sistema diretamente no celular para:

Criar uma conta

Registrar uma nova denúncia

Adicionar informações como localização, descrição e categoria

Enviar a denúncia para o sistema

4.4. Coleta de Feedback

Após o teste, foi realizada uma conversa estruturada para registrar opiniões, dificuldades e sugestões.

5. Evidências da Validação

As evidências foram armazenadas na pasta validation/evidence/ e incluem:

Prints da reunião apresentando o sistema

Todos os participantes autorizaram o uso das imagens exclusivamente para fins acadêmicos.

6. Feedback Coletado

Os feedbacks estão registrados em arquivos individuais em validation/feedback/.

Principais pontos identificados:

Pontos Positivos

Interface simples e fácil de entender.

Rápido para registrar uma denúncia.

Utilidade real para situações vividas no bairro.

Dificuldades observadas

Alguns termos podiam ser mais simples.

Pediram inclusão de categorias adicionais (“Terreno baldio”, “Queima de lixo”).

Sugestões adicionais

Criar uma opção para acompanhar o andamento da denúncia.

Permitir fotos para denúncias graves.

Integrar com WhatsApp para compartilhar denúncia.

8. Conclusão

A validação comprovou que o sistema “Vizinhança Limpa” é funcional, intuitivo e relevante para moradores que lidam com problemas ambientais locais.
Os participantes entenderam rapidamente como utilizar a ferramenta e reconheceram sua utilidade prática no dia a dia da comunidade.

O retorno coletado foi essencial será essencial para aprimorar a experiência do usuário e reforçar o impacto social do projeto.
